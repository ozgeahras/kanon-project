## Run Locally

Clone the project

```bash
  git clone https://github.com/ozgeahras/kanon-project.git

```

Go to the project directory

```bash
  cd ..kanon-project
```

Install dependencies

```bash
npm install
```

Start the app

```bash
npm run dev
```
